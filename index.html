Functioning of the Calculator

1. Input Collection: The calculator takes three inputs from the user: age, gender, and coverage amount.

2. Base Premium Calculation: A base premium is calculated as 5% of the coverage amount.

3. Age Adjustment:

  • If the user is under 25 years old, the premium is increased by 50%.
  • If the user is over 60 years old, the premium is doubled.

4. Gender Adjustment:

  • If the user selects "male", the premium is increased by 10%.
  • If the user selects "female", the premium is increased by 5%.

5. Premium Display: The final premium amount is calculated based on the adjustments and is displayed to the user in a visually highlighted area.
